# 🍔 Mordelón — Generador de Historias

App para generar imágenes de Instagram (Story y Post) para Mordelón.

## Instalación local

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Subí esta carpeta a un repositorio en GitHub
2. Entrá a [vercel.com](https://vercel.com) y conectá tu cuenta de GitHub
3. Importá el repositorio
4. Vercel detecta Vite automáticamente — no hace falta configurar nada
5. Click en **Deploy** y listo

## Uso

- Elegí un **preset** (Lunes, Viernes, Promo, etc.)
- Subí tu **foto de fondo**
- Opcionalmente subí un **logo**
- Ajustá textos, paleta de colores y oscuridad
- Descargá en **HD** (1080×1920 para Story, 1080×1080 para Post)
